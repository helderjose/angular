Tutorial

  • Crie o module cursos:
    ng g m cursos
    ng g module cursos
    Será criada uma pasta cursos com o cursos.module.ts

  • Crie o componente cursos
    ng g c cursos
    Será criados os arquivos do component na pasta cursos e o arquivo app.module.ts será atualizado.
    No arquivo app.module.ts remova o import referente ao 'CursosComponent' e remova-o do metadado 'declarations'.

  • cursos.module.ts
    Import o 'CursosComponent'.
    Insira 'CursosComponent' no metadado 'declarations'.
    Aqui o cursos.component já pertence ao cursos.module.

    Adicione o metadado 'exports' e coloque o 'CursosComponent' nele.

  • app.module.ts
    Usando o 'cursos.module' no app.module.ts
    Import o 'CursosModule'.
    Use o 'CursosModule'' no metadado 'declarations'.
    Use a tag app-cursos no app.component.html.

  • Obs: Ao importar um module, posso usar tudo que ele expõe em qualquer componente que esteja declarado.

  • Criar o component curso-detalhe
    ng g c cursos/curso-detalhe
    Como esse componente foi criado dentro da pasta 'cursos', ele será automanticamente declarado no cursos.module.ts
    Como o 'CursoDetalheComponent' foi declarado no 'cursos.module.ts', você pode usar o seletor 'app-curso-detalhe' de 'CursoDetalheComponent' no cursos.component.html.

  • Não consigo usar o selector de 'CursoDetalheComponent' neste arquivo porque não exportei-o no curos.module.ts
