View Encapsulation

Pesquise por Shadow DOM. Angular simula um shadown DOM quando usamos View Encapsulation. 

    No inspetor de elementos podemos ver a hierarquia no css dependendo do modo que
    usamos no ViewEncapsulation.

    ViewEncapsulation.Emulated
        Angular emulates Shadow DOM. So it understands that most browsers out there don't support
        Shadow DOM, so it has it's own trick to apply it's scope to our styles.
        Esse é o default.
    
    encapsulation: ViewEncapsulation.Native
        Native viwe encapsulation which uses shadow DOM in the browser.

    encapsulation: ViewEncapsulation.None
    here we're not going to have view encapsulation. So the styles defined here will leak outside this template. In other words, when we go back to the browser, both our icons are going to be red.
